## Hello

I've been involved with the design, development and deployment of web applications and web "stuff" since the late 1990s. I was full stack before that was a thing. I'm currently working for myself under the guise of [Trees & Code](https://github.com/organizations/trees-and-code). I spent six years in executive roles between 2016-2022 it was a blast, and they tell me I nailed it, but I decided it wasn't for me so in 2022 I took some time out and did two things. One, I went back to writing code as my primary practice (_rather than doing executive things_) and two, I spent some time getting trained in arboriculture and woodland management. Since then I've been spending my time writing code, deploying stuff and taking care of trees. 


## Project Learn

I'm always learning and so I've got an [active Github project](https://github.com/users/treejamie/projects/2) where you can see what I'm up to. I've tried to keep the project in sprints so you can see that "know what I'm doing" when working in as a software person in a team of people.

Thanks to the tree work I do, I've got a really nice balance in my life now and it's helped me focus in on a goal - to be the best programmer I can be. I'm concentrating on learning Go, Typescript, Rust and understanding web assembly.

I'm most solid in Python (_and Django_) which I've used since 2007 and Elixir (_ and phoenix_) since 2017. However, in tech world things move fast and so I'm brushing up on everything.
